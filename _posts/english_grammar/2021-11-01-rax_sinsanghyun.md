- 어근은 **힘, 위치, 방향** 속성을 잘 생각하며 파악해본다.

<br>

- `ab-`  : 이탈
- `e-`  : (밖으로) 나온 상태
- `in-` : 둘러싼 것
- `inter-` : 사이에 있다
- `il-`, `ir-`, `im-` : 부정
- `de-` : 하락, 떨어진, 완전히
- `as-` : 나아가는(?)
- `hypo-` : 아래
- `re-` : "다시" again이 핵심
  - $\oplus$뒤로, $\oplus$원래대로
- `en-` : 상태로 만듦, 둘러싼
- `dis-` : 이탈 
  - $\oplus$부정, $\oplus$이탈이 가속
- `com-` : 어깨동무, 
  - $\oplus$함께
- `pre-` : 미리, 조기, 그 이전의
- `pro-` : 앞으로
- `per-` : 완전히
- `sur-` : 위에
- `trans-` : 가로 질러
- `sub-` : 아래에(서)
- `su-` : 아래로
- `ob-` : to
- `un-` : 부정
- `out-` : 밖으로 나온 상태, 밖으로 나가는 상태
- `a-` : ~에, ~로, (=on)
  - $\oplus$강조
- `dia-` : 사이에(between), 가로질러(across)
- `mis-` : 잘못된(bad, wrong)
- `se-` : 떨어져(apart)
- `up-` : 위로
- `ab-` : 떨어져(away from), 강조(intensive)
- `auto-` : 자신의, 스스로의(self)
- `tele-` : 멀리(far off)
- `ana-` : ~내내(throughout)
- `para-` : against
- `inter-` : 사이, 중간, 도중 (among)
