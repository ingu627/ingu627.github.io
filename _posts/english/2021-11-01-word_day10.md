## 기본패턴

- 명사/동사 + -ly = -ly (형용사)
- 형용사 + -ly = -ly (부사)

<br>
<br>

## vive : 살다

- **live** : 살다, 생방중인, 살아있는
  - lively : 활기넘치는
  - alive : 살아 있는
  - outlive : ~보다 오래살다
  - **life** : 삶, 인생
- **vivid** : 생생한
- **survive** : 생존하다
  - survival : 생존
- **revive** : 되살아나다
  - revival : 부활
- **cf**
  - lifetime : 일생
  - lifestyle : 삶의 방식
  - life span : 수명

<br>
<br>

## love : 사랑하다, 사랑 

- **lovely** : 사랑스러운, 멋진 
  - beloved : 아주 사랑하는, 인기많은, 사랑하는 사람
  - lover : 애인, 애호가
- **believe** : 믿다, 생각하다
  - believable : 믿을 수 있는
  - unbelievable : 믿을 수 없는
  - belief : 믿음
- **cf**   
  - ~인체 하다

<br>
<br>

## free : 자유로운, 무료의, 자유롭게 하다

- **friend** : 친구
  - friendly : 우호(친화)적인
  - unfriendly : 비우호적인
  - environment-friendly : 환경친화적인
  - business-friendly : 기업친화적인
  - friendship : 우정
- **freedom** : 자유
  - freely : 자유롭게
- **frank** : 솔직한
  - frankly : 솔직히
- **cf**
  - free from : ~이 없는 
  - free of : ~무료의 
  - for free : 무료로

<br>
<br>

## see : 보다, 알다

- **sight** : 시야, 시력, 광명
  - sightly : 보기좋은 
  - unsightly : 보기흉한
  - **sightseeing** : 관광
  - insight : 통찰력
  - hindsight : 뒤늦게 앎(깨달음)
  - near-sighted : 근시의 
  - far-sighted : 원시의, 장기적인 
- **foresee** : 예견하다
  - foresight : 선경지명
- **oversee** : 감독하다
- **cf**
  - in sight : 보이는
  - out of light : 안 보이는
  - catch sight of : ~을 언뜻 보다
  - lose sight of : 놓치다, 안 보이다

<br>
<br>

## di : 하루, 낮 

- **day** : 하루, 낮, 시기 
  - daily : 매일의, 일상적인
  - midday : 정오
  - nowadays : 요즈음에
- **daybreak** : 새벽
- **diurnal** : 주행성의
- **diary** : 일기
  - dial : 전화를 걸다
- **diet** : 식단, 다이어트
  - dietary : 규정식의 
- **cf**
  - day off : 쉬는 날
  - one day : 어느날, 언젠가
  - in a day : 하루아침에, 하루 사이에 
  - day to day : 일상의 
  - these days : 요즈음

<br>
<br>

## noct : 밤

- **night** : 밤
  - nightly : 밤의
  - nightmare : 악몽
  - midnight : 자정
  - tonight : 오늘밤, 오늘밤에
  - overnight : 밤사이에, 갑자기, 야간의
- **nocturnal** : 야행성의

<br>

- **week** : 주
  - weekly : 매주의
  - biweekly : 격주의
  - weekday : 평일
  - weekend : 주말

<br>

- **moon** : (밤의) 달
  - month : 달 
  - monthly : 매월의 
  - bimonthly : 격월의 

<br>
<br>

## like : 같은, ~와 같은, ~처럼, 좋아하다

- **likely** : ~할 것 같은, ~하기 쉬운
  - likelihood : 가능성
  - unlikely : ~할 것같지 않은, 믿음직하지 않은
  - alike : 비슷한, 똑같이
  - unlike : 다른, ~와 달리
  - dislike : 싫어하다
- **likewise** : 마찬가지로, 비슷하게
- **cf**
  - as likely as not : 아마
  - most likely : 아마

<br>
<br>

## one : 하나의, 하나

- **alone** : 혼자인, 홀로, 단지 
  - lone : 혼자인
  - lonely : 외로운
  - loneliness : 외로움
  - only : 오직, 유일한

<br>

- **uni** : 하나
  - unique : 독특한
  - unite : 연합(단합)하다
  - unity : 통일(성), 단합
  - unify : 통합하다 
  - union : 결합, 단체, 노조
- **none** : 아무도 [조금도] ~ 않다
  - nonetheless : 그렇기는 하지만, 그렇더라도

<br>
<br>

## die : 죽다, 사라지다

- **dead** : 죽은, (the) 죽은 사람들
  - deadly : 치명적인
  - deadline : 기한
  - death : 죽음
  - die out : 멸종하다
  - die down : 줄어들다 

<br>

- **low** : 낮은, 낮게
  - lowly : 낮은, 하찮은
  - lower : 보다, 낮은, 낮추다
  - below : 위에

<br>
<br>

## kin : 친족

- **kind** : 친절한, 종류, 어떤(of)
  - kindly : 친절하게, 제발
  - kindness : 친절
  - unkind : 불친절한
  - unkindly : 불친절하게
- **kindergarden** : 유치원
- **kinship** : 친족관계
  - akin : 유사한
- **cf**
  - mankind : 인류
  - humankind : 인류

<br>
<br>

## peri[pert] : 시도하다

- **expert** : 숙련된, 전문의, 전문가
  - expertly : 능숙하게
  - expertise : 전문지식
  - inexpert : 미숙한
- **experience** : 경험, 겪다
  - experienced : 능숙한, 경험있는
  - inexperienced : 미숙한
- **experiment** : 실험, 실험하다
  - experimental : 실험적인
  - experimentally : 실험적으로
