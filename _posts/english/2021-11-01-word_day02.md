## 기본패턴

- **-ic** : 명사 (성질, 성향)
- **-ic + -al = -ical** : 형용사(~의, ~인)
- **-ic + -al + -ly = -ically** = 부사(~하게)

<br>

- **-ic** : 형용사 (~의, ~인)
- **-ic + -al = ical** : 형용사 (~의, ~인)
- **-ic + -ally = -ically** : 부사 (~하게)

## log[logue] : 말

- **logic** : 논리
    - logical : 논리적인
    - logically : 논리적으로
- **illogic** : 비논리
    - illogical : 비논리적인
    - illogically : 비논리적으로
- **apology** : 사과
    - apologetic : 사과하는
    - apologize : 사과하다
- **cf**
    - prologue : 서두, 프롤로그
    - epilogue : 끝맺는 말
    - monologue : 독백, 1인극

## crite : 판단(결정)하다

- **critic** : 비평가
    - critical : 비판적인, 결정적인
    - critically : 비판적으로
    - criticize : 비판하다
    - criticism : 비판
- **criterion** : (판단, 결정의) 기준
- **crisis** : 위기
- **hypocrite** : 위선자
    - hypocritical : 위선적인
    - hypocritically : 위선적으로
    - hypocrisy : 위선

## med : 취료하다

- **medic** : 위생병

    - medicine : 약
- **meditate** : 명상하다
    - meditation : 명상
- **remedy** : 치료(약), 해결(개선)책 
- **cf**
    - **medi** : 중간

    - media : (방송)매스미디어
    - medieval : 중세
    - immediate : 즉각적인
    - immediately : 즉각적으로 = 즉시
