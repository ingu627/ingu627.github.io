## 기본패턴

- 어근/단어 + -ary = -ary (형용사)
- ary(형용사) + -ly = -arily (부사)
- -ery는 대부분 명사 접미사로 사용

<br>
<br>

## cess : 포기하다, 중지하다

- **necessary** : 필요한, 필연적인, 포기할 수 없는
  - necessarily : 반드시, 필연적으로
  - necessity : 필요(성)
- **unnecessary** : 불필요한
  - unnecessarily : 불필요하게
- **incessant** : 끊임없는
  - incessantly : 끊임없이
- **cease** : 중지하다
  - ceasefire : 휴전
  - ceaseless : 끊임없는
  - decease : 죽음, 사망하다

<br>
<br>

## vol[volunt] 의지

- **voluntary** : 자발적인
  - voluntarily : 자발적으로
  - involuntary : 본의 아닌
  - involuntarily : 본의아니게
- **volunteer** : 자원봉사자, 자원하다
- **will** : 의지, 유언
  - willing : 기꺼이하는
  - willingly : 기꺼이
  - willingness : 의향
  - unwilling : 꺼리는, 마지못해 하는
  - unwillingly : 꺼리게, 마지못해서
- **goodwill** : 친선, 호의

<br>
<br>

## ordin : 순서

- **ordinary** : 평범한, 보통의
  - ordinarily : 보통, 대개
- **extraordinary** : 비범한, 엄청난
  - extraordinarily : 몹시, 엄청나게
- **coordinate** : 조정하다, 좌표
- **subordinate** : 부하의, 종속된, 부하
- **order** : 순서, 질서, 명령, 주문(서), 명령하다, 주문하다
  - orderly : 정돈된
  - disorder : 부질서, 엉망
- **cf**
  - out of order : 고장난
  - in order : 제대로 된, 순번대로

<br>
<br>

## custom : 관습, 습관

- **customary** : 관례적인
  - customarily : 관례상
- **customer** : 손님, 고객
  - customize : 주문받다, 만들다
- **customs** : 세관, 관세
- **accustomed** : 익숙한 (to)
  - unaccustomed : 익숙하지 않은
- **costume** : 복장, 의상

<br>
<br>

## im(em) : 복사하다

- **image** : 이미지
  - imagine : 상상하다
  - imaginary : 가상의, 상상의
  - imagination : 상상(력)
  - imaginative : 상상력이 풍부한 
  - imaginable : 상상할 수 있는
  - unimaginable : 상상할 수 없는
- **imitate** : 모방하다
  - imitation : 모방, 모조(품) 
  - emulate : 흉내내다, 경쟁하다

<br>
<br>

## liter : 글자

- **literary** : 문학의
  - literature : 문학
- **literate** : 읽고 쓸줄 아는
  - literacy : 읽고 쓰는 능력
  - illiterate : 문맹의
  - illiteracy : 문맹  
- **letter** : 글자, 편지
- **literal** : 글자 그대로의
  - literally : 글자 그대로
- **obliterate** : 제거하다, 지우다
  - obliteration : 제거, 말소

<br>
<br>

## temper : 시간, rote : (기계적 반복의) 암기

- **temporary** : 일시적인, 임시의
  - temporarily : 일시적으로, 임시로
- **contemporary** : 동시대의, 현대의
  - contemporarily : 동시대에, 현대에
- **tempo** : 박자

<br>

- **rotary** : 회전식의, 로터리
- **rotate** : 회전하다, 교대근무하다
  - rotation : 회전, 자전, 교대
- **route** : 길, 경로
  - routine : 일상적인, 일과

<br>
<br>

## sum : 액수, 통합, sal : 소금

- **sum up** : 요약하다
  - summary : 요약
  - summarize : 요약하다
- **summit** : 정상, 절정
- **secret** : 비밀의, 비밀
  - secretary : 비서, 장관
  - secretly : 비밀리에

<br>

- salary : 봉급
- salad : 샐러드
- sult : 소금 
- salty : 짠

## mechanic : 정비공, scene : 혀장, 장면, brew : 양조하다

  - mechanical : 기계의
