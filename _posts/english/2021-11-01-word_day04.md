## 기본패턴

- **-ent** : 형용사(~한, ~하는)
- **-ent + -ly = -ently** : 부사(~하게, ~로)
- **-ence = -ency** : 명사(~한 상태, 행동)

<br>
<br>

## obey : 복종하다, 따르다

- **obedient** : 복종하는
    - obediently : 고분고분하게
    - obedience : 복종
- **disobey** : 불복종하다
    - disobedient : 불복종하는
    - disobediently : 반항적으로
    - disobedience : 불복종, 반항
- **obeisance** : 경의

<br>
<br>

## depend : 의지하다, 달려있다(on)

- **dependent** : 의존하는, ~에 달려있는
    - dependently : 의존하여
    - dependence : 의존
    - independently : 독립적으로
    - independence : 독립
- **independent** : 독립된
- **depending on** : ~에 따라
- **dependable** : 신뢰할 수 있는
    - dependability : 신뢰성

<br>
<br>

## compete : 경쟁하다, 겨루다

- **competent** : 유능한
    - competently : 유능하게
    - competence : 능력
- **incompetent** : 무능한
    - incompetently : 무능하여
    - incompetence : 무능
- **competition** : 경쟁, 대회
    - competitor : 경쟁자
    - competitive : 경쟁을 하는
    - competitively : 경쟁적으로

<br>
<br>

## differ : 다르다

- **different** : 다른
    - differently : 다르게
    - difference : 차이
- **indifferent** : 무관심한
    - indifferently : 무관심하게
    - indifference : 무관심
- **cf**
    - differential : 차등하는, 차이, 차등
    - differentiate : 구별(구분)하다
    - differentiation : 구별

<br>
<br>

## viol : 부수다

- **violent** : 폭력적인
    - violently : 폭력적으로, 격렬하게
    - violence : 폭력
- **non-violent** : 비폭력적인
    - non-violence : 비폭력
- **violate** : 위반(침해)하다
    - violation : 위반
    - violator : 위반자
    - violative : 위반하는

<br>
<br>

## fide[feder] : 믿다

- **confide** : 신뢰하다, 비밀을 털어놓다
    - **confident** : 자신있는, 확신하는
    - confidently : 자신있게
    - confidence : 자신감, 신뢰
- **confidential** : 비밀의
    - confidentially : 은밀히
- **federal** : 연방의
    - federally : 연방제로
- **confederate** : 연방의, 동방의
    - confederation : 연합, 동맹

<br>
<br>

## sist : 서다(세우다)

- **persist** : 고집하다, 지속하다
    - persistent : 고집하는, 지속하는
    - persistently : 고집스럽게, 끊임없이
    - persistence : 고집, 지속
- **insist** : (사실들을)주장하다
    - insistent : 주장(고집)하는, 계속되는
    - insistently : 끈덕지게
    - insistence : 주장
- **consist** : 구성되다(of), ~에 있다(in)
    - consistent : 일관된
    - consistently : 일관되게
    - consistency : 일관성 
    - inconsistent : 일관성 없는, 모순된
    - inconsistently : 일관성없이

<br>
<br>

## fice[fici] : 만들다, 하다

- **suffice** : 충분하다
    - sufficient : 충분한
    - sufficiently : 충분히
    - sufficiency : 충분한 양, 충분
    - insufficient : 불충분한
- **deficiency** : 결핍, 부족
- **deficit** : 적자
- **efficient** : 유능한, 효율적인
    - inefficient : 비효율적인
    - efficiently : 효율적으로
    - efficiency : 효율(성)

<br>
<br>

## cide : 떨어지다

- **accient** : 사고
    - accidental : 우연한
    - accidentally : 우연히, 실수로
    - by accident : 우연히 
- **incident** : 사건
    - incidential : 부수적인
    - incidentally : 그런데
- **coincide** : 일치하다, 동시에 일어나다
    - coincident : 일치하는
    - coincidence : 일치, 우연

<br>
<br>

## ori : 뜨다

- **orient** : 동양, 지향하게 하다, 적응하다
    - oriental : 동양(인)의
    - oriented : 지향적인
    - orientation : 지향, 오리엔테이션
- **origin** : 기원, 출신
    - original : 독창적인, 원래의, 원본
    - originally : 원래, 본래
    - originality : 독창성
- **aborigine** : 원주민
    - aboriginal : 원주민의
    - originate : 시작되다, ~에서 비롯되다

<br>
<br>

## side : 앉다

- **reside** : 거주하다
    - resident : 거주자, 주민
    - residence ; 거주(지)
    - residential : 거주의
- **preside** : 주재(주최)하다
    - president : 대통령, 회장
    - presidency : 대통령(회장)직
    - presidential : 대통령의
- **sit** : 앉다
    - seat : 자리, 좌석, 앉히다
    - seatbelt : 안전벨트
