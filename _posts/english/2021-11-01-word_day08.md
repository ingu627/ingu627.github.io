## 기본패턴

- 명사/동사 + -ar = -ar
- -le로 끝나는 단어 + -ar = -ular

<br>
<br>

## Popule : 사람들

- **popular** : 인기 있는
  - popularly : 대중적으로, 널리
  - popularity : 인기
- **unpopular** : 인기 없는
- **people** : 사람들
- **populate** : 살다, 거주하다.
  - population : 인구
  - overpopulation : 인구과잉 
- **populous** : 인구가 많은
  - populism : 포퓰리즘

<br>
<br>

## regule : 규칙

- **regular** : 규칙적인, 정규적인
  - regularly : 규칙적으로
- **irregular** : 불규칙적인, 비정규적인
  - irregularly : 불규칙적으로
- **regulate** : 규제하다, 조절하다
  - regulation : 규제, 규정
  - regulator : 규제기관, 조절기
  - regulatory : 규제의
- **rule** : 규칙, 지배하다
  - ruler : 지배자, 자
  - rule out : 배제하다

<br>
<br>

## circule : 원(형)

- **circular** : 원형의, 순환적인
  - circularly : 원형으로
- **circulate** : 순환하다, 유표되다
  - circulation : 순환, 유통
- **circle** : 원(형)
  - encircle : 에워(둘러)싸다
  - semicircle : 반원
- **cycle** : 주기, 순환하다
  - bicycle : 자전거
  - motorcycle : 오토바이
  - recycle : 재활용하다
  - recycling : 재활용

<br>
<br>

## ang : 구부리다

- **angle** : 각
  - angular : 각진
- **triangle** : 삼각형
  - triangular : 삼각형의
  - rectangle : 직사각형
  - rectangular : 직사각형의
- ankle : 발목
- **anchor** : 닻, 앵커, 고정시키다

<br>
<br>

## part : 부분, 역할

- **particle** : 소량 $\Rightarrow$ particle : 소량, 입자
- **particular** : 특정한, 특별한
  - particularly : 특히
  - in particular : 특히
- **as part of** : ~의 일부로, ~의 일환으로
- **partial** : 부분적인
  - impartial : 편견없는, 공편한
- **participate** : 참가하다
  - participation : 참가
  - participant : 참가자
- **cf**
  - partition : 분할
  - apart : 따로, 떨어져
  - compartment : 구획, 칸
  - apart from : ~을 제외하고, ~뿐만 아니라

<br>

## spect : 보다

- **spectacle** : 장관, 광경
  - spectacles : 안경
  - spectacular : 장관을 이루는
  - spectacularly : 장대하게, 굉장히
  - spectator : 관중
- **mole** : 덩어리
  - molecule : 분자
  - molecular : 분자의
  - demolish : 파괴하다, 철거하다
  - demolition : 파괴, 폭파
- mole : 작은 점, 사마귀

<br>

## simil[simul] : 같은

- **similar** : 유사한, 비슷한
  - similarly : 유사하게
  - similarity : 유사점
  - dissimilar : 다른
- **seem** : ~인 것 같다, ~처럼 보이다
- **same** : (똑)같은, 똑같은 것
- **simulate** : 가장하다, 모의실험하다
  - simulation : 가장, 모의실험
- **simultaneous** : 동시의
  - simultaneously : 동시에
- **cf**
  - all the same : 그래도, 역시
  - at the same time : 동시에
  - in the same way : 같은 식으로
