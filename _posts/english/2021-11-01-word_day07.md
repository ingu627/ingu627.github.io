## 기본패턴 

- **-ous**는 **-ful**과 비슷

<br>
<br>

## mount : 오르다, 올라타다

- **mountain** : 산
  - mountainous : 산악의
- **amount** : 양, 액수
  - amount to : (총액 등) ~에 이르다
- **mound** : 둔덕, (야구)마운드
- **surmount** : 극복하다
  - surmountable : 극복할 수 있는
  - insurmountable : 극복할 수 없는
- **paramount** : 최고의, 가장 중요한

<br>
<br>

## (o)nym : 이름

- **anonym** : 익명, 가명
  - anonymous : 익명의, 작가 불명의
- **antonym** : 반의어
- **synonym** : 유의어
- **name** : 이름, 명명하다
- **nomin** : 이름
  - nominal : 명목상의
  - nominate : 지명하다, 추천하다
  - nomination : 지명, 추천
  - nominee : 지명된 사람은
- **noun** : 명사
  - pronoun : 대명사
  - renoun : 명성
  - renouned : 명성 있는, 유명한

<br>
<br>

## cour[cord] : 마음

- **courage** : 용기
  - courageous : 용감한
  - courageously : 용감하게
- **encourage** : 격려하다, 권장하다
  - encouragement : 격려
- **discourage** : 낙담시키다
- **core** : 핵심, 속
- **accord** : 일치, 협정
  - according to : ~에 따르면, ~에 의하면
  - accordingly : 따라서
- **record** : 기록, 기록(녹음)하다
  - recording : 녹음(한것)
- **cf**
  - in accord with : ~에 일치하여
  - of one's own accord : 자발적으로, 스스로

<br>
<br>

## fa : 말하다

- **fame** : 명성
  - famous : 유명한
  - famously : 유명하게
  - infamous : 악명높은
- **infant** : 유아, 유아의
  - infancy : 유아
- **fate** : 운명
  - fateful : 운명적인
  - fatal : 치명적인
  - fatally : 치명적으로
  - fatality : 사망자, 치사율
- **fable** : 우화
  - fabulous : 멋진, 엄청난
  - fabulously : 엄청나게

<br>
<br>

## fail : 실패하다, 못하다, 낙제

- **fallacy** : 오류, 잘못된 생각
  - fallacious : 잘못된
- **failure** : 실패하다
- **without fail** : 틀림없이, 반드시
- **false** : 거짓의, 잘못된
  - falsehood : 거짓
- **fault** : 잘못, 결함
  - default : 채무불이행
  - faulty : 흠(결함)이 있는

<br>
<br>

## ar : 적합하다

- **harmony** : 조화
  - harmonious : 조화로운
  - harmoniously : 조화롭게
- **harmonize** : 조화롭게 하다
- **disharmony** : 부조화, 불화
- **art** : 예술, 미술
  - artist : 예술가
  - artistic : 예술의
  - artisan : 장인
- **artifact** : 공예품
  - artificial : 인공적인, 인조의
  - artificially : 인공적으로

<br>
<br>

## myst : 닫다

- **mystery** : 미스터리, 신비
  - mysterious : 신비한
  - mysteriously : 신비롭게
- **mystic** : 신비주의자, 영적인
  - mystical : 신비주의의
  - mysticism : 신비주의
- **myopia** : 근시
  - myopic : 근시적인
- **mute** : 무언의, 말을 못하는
- **cf**
  - optic : 눈(시력)의
  - optical : 광학의, 시각의
  - optician : 안경사

<br>
<br>

## it : 가다

- **ambition** : 야망
  - ambitious : 야심있는
  - ambitiously : 야심차게 
- **exit** : 출구, 퇴장, 나가다
- **initial** : 처음(시작)의, 머리글자
  - initially : 처음에
  - initiate : 시작(가입)하다
  - initiation : 가입, 입문
  - initiative : 주도, 계획
- **cf**
  - transit : 통과
  - transition : 과도, 변천

<br>
<br>

## fact : 만들다, 하다

- **infect** : 전염(감염)시키다
  - infection : 전염, 감염
  - infectious : 전염되는
- **affect** : ~에 영향을 미치다
  - affection : 애정
  - affectionate : 애정어린
  - affectionately : 애정을 담고
- **perfect** : 완벽한, 완전한
  - perfectly : 완벽하게, 완전히
  - perfection : 완벽, 완성
  - imperfect : 불완전한
- **effect** : 효과, 영향, 결과
  - effective : 효과적인
  - effectively : 효과적으로
  - in effect : 시행중인, 사실상

<br>
<br>

## spect[spic] : 보다

- **suspect** : 의심하다, 혐의자
  - suspicion : 의심, 혐의
  - suspicious : 의심스러운
- **prospect** : 가망, 전망
  - prospective : 유망한, 예비의
- **respect** : 존경, 존경하다
  - respectful : 공손한, 예의바른 
  - respectable : 훌륭한, 존경할만한 
- **disrespect** : 무례
  - disrespectful : 무례한
- **cf**
  - in respect of : ~에 관하여
  - respectively : 각각
  - respective : 각각의
  - irrespective : 관계없이

<br>
<br>

## lig : 읽다

- **religion** : 종교
  - religious : 종교의
  - religiously : 종교적으로, 어김없이
- **negligent** : 부주의한, 태만한
  - negligently : 부주의하게, 태만하게
  - negligence : 부주의, 태만
- **diligent** : 근면한
  - diligently : 근면하게
  - diligence : 근면
- **inteligent** : 총명한, 지적인
  - inteligently : 총명하게
  - inteligence : 지능, 정보
