## 기본패턴 

- `-t` (동사)
- `-t` + `-ion` = `-tion` (명사(행동)) 
- `-t` + `-or` = `-tor` (명사(사람, 사물))  
- `-t` + = `-ive` = `-tive` (형용사)

<br>
<br>

## ag : 행동하다, 몰다

- **agent** : 대리인, 요원, 행위자
    - agency : 대리점, 매개, 기관
- **agile** : 민첩한, 신속한
    - agility : 민첩함
- **agitate** : 선동하다, 동요시키다
    - agitation : 선동, 동요
- **agony** : 고통, 고민
    - agonize : 고뇌하다, 고민하다
    - antagonist : 적대자

<br>
<br>

## act: 행동하다, 연기하다

- **action** : 행동(실행), 조치
    - active : 활동적인, 적극적인
    - actively : 활발하게, 적극적인
    - actor : 배우
    - activity : 활동
    - activate : 활성화하다
    - actress : 여배우
- **interact** : 상호작용하다
    - interaction : 상호작용
    - interactive : 상호작용하는, 대화형의
- **exact** : 정확한
    - exactly : 정확히
- **cf**
    - take action : 조치를 취하다

<br>
<br>

## drag : 끌다, 끌고가다, 장애물

- **draw** : 끌다, 그리다
    - drawing : 그림, 소모
    - drawer : 서랍
    - drawback : 결점, 단점
- **withdraw** : 인출하다, 철수하다
    - withdrawal : 인출, 철수
- **trace** : 흔적, 자취, 흔적을 쫓다
    - track : 자국, 트랙, 길, 추적하다
- **trek** : 고된여행, 고된 여행을 하다
    - trigger : 방아쇠, 유발하다

<br>
<br>

## tract : 끌다

- **attract** : (관심, 미움을) 끌다
    - attraction : 매력(끌림), 명소
    - attractive : 매력적인
    - attractively : 매력적으로
- **extract** : 추출하다, 추출물
    - extraction : 추출
    - extractor : 추출기
- **subtract** : 빼다
    - subtraction : 빼기
- **distract** : 산만하게 하다
    - distraction : 산만함, 집중방해

<br>
<br>

## seg[sect] : 자르다

- **segment** : 부분, 분할하다
    - saw : 톱
- **section** : 부분, 구획
    - sector : 부채꼴, 부분, 분야
- **dissect** : 해부(분해)하다
    - dissection : 해부
- **intersection** : 교차로 
- **insect** : 곤충

<br>
<br>

## lect: 고르다

- **collect** : 모으다, 수집하다
    - collection : 모음, 수집품
    - collector : 수집가
    - collective : 집단의, 집약적인
    - collectively : 집단적으로
- **select** : 선택하다
    - selection : 선택
    - selector : 선택자, 선택장치
    - selective : 선택적인
- **elect** : 선거(선출)하다
    - election : 선거
    - elector : 선거인
    - elective : 선거의(선택의), 선택과목
- **intellect** : 지적능력, 지성
    - intellectual : 지적인
- **neglect** : 방치하다, 무시하다, 방치, 소홀
- **cf**
    - recollect : 회상하다
    - recollection : 회상

<br>
<br>

## rect: 똑바로 세우다

- **correct** : 수정(교정)하다, 올바른, 정확한
    - correction : 수정, 정정
    - corrector : 수정(교정)자
    - correctly : 올바르게
- **incorrect** : 부정확한
    - incorrectly : 부정확하게
- **direct** : 향하게 하다, 지시하다, 직접적인
    - direction : 방향, 지시
    - director : (영화)감독, 임원
    - directory : 명단, 목록
    - directly : 똑바로, 즉시
- **indirect** : 간접적인
    - indirectly : 간접적으로
- **cf**
    - erect: 건립하다, 세우다, 똑바로 선

<br>
<br>

## ser[sert] : 높다

- **assert** : 주장하다
    - assertion : 주장
    - assertive : 독단(적극)적인
- **insert** : 삽입하다
    - insertion : 삽입
- **exert** : (영향) 행사하다, 노력하다
    - exertion : 행사, 노력
- **desert** : 버리다, 사막
    - deserted : 버려진, 떠난
    - desertion : 버림, 유기
- **cf**
    - serial : 연재물, 연속적인
    - series : 연속, 시리즈

<br>
<br>

## gest : 옮기다

- **suggest** : 제안(제시)하다
    - suggestion : 제안
- **congest** : 혼잡하게 하다
    - congestion : 혼잡
- **digest** : 소화하다, 요약하다, 요약
    - digestion : 소화
    - digestive : 소화의
    - indigestion : 소화불량
    - indigestive : 소화불량의
- **register** : 등록하다, 명부
    - registration : 등록, 접수

<br>
<br>

## ject: 던지다

- **eject** : 내쫓다, 방출하다
    - ejection : 방출
- **inject** : 주입(주사)하다
    - injection : 주입, 주사
- **reject** : 거절(거부)하다
    - rejection : 거절, 거부
- **object** : 반대하다(to), 물체, 목적(어)
    - objection : 반대, 이의
    - objective : 객관적인, 목표
    - objectivity : 객관성
- **subject** : 종속시키다(to), 대상, 주제(주어), 과목
    - subjective : 주관적인
    - subjectivity : 주관성
- **cf**
    - be subject to : ~을 받기 쉬운

## dict[dicate] : 말하다

- **addict** : 중독시키다, 중독자
    - addiction : 중독
    - addictive : 중독성의
- **predict** : 예측하다 
    - prediction : 예측
- **contradict** : 부정(반박)하다, 모순하다
    - contradiction : 부정, 모순
- **dedicate** : 바치다, 전념하다
    - dedication : 전념, 헌신
- **indicate** : 가리키다, 나타내다
    - indication : 지시, 표시
    - indicator : 지표, 표시기
- **dictate** : 명령(구술)하다
    - dictation : 받아쓰기, 구술
    - dictator : 독재자
    - dictatorship : 독재
