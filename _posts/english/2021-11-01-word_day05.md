## 기본패턴

- **-ant** : 형용사(~한, ~하는)
- **-ant + -ly = -ently** : 부사(~하게, ~로)
- **-ance = -ancy** : 명사(~한상태, 행동)

<br>
<br>

## port : 항구

- **import** : 수입하다, 수입
    - important : 중요한
    - importantly : 중요하게
    - importance : 중요성
    - unimportant : 중요하지 않은
- **export** : 수출하다, 수출
- **cf**
    - passport : 여권
    - airport : 공항
    - opportune : (시기가)적절한
    - opportunity : 기회

<br>
<br>

## sign : 표시, 징후, 서명하다

- **signify** : 의미하다, 중요하다
    - significant : 의미있는, 중요한
    - significantly : 상당히, 현저하게
    - significance : 의미, 중요성
- **insignificant** : 사소한
    - insignificance : 사소함
- **signature** : 서명
- **assign** : 할당하다, 지정하다
    - assignment : 과제, 숙제
- **design** : 디자인, 설계, 설계(고안)하다
    - designer : 디자이너
    - designate : 지정하다
    - designation : 지정

<br>
<br>

## tol[tal] : 올리다

- **toler** : 참다
    - tolerant : 관대한, 내성있는
    - tolerantly : 관대하게
    - tolerance : 관용, 내성
    - intolerant : 편협한
- **tolerable** : 참을 수 있는
    - intolerable : 참을 수 없는
- **tolerate** : 참다, 용인하다
- **toll** : 통행료
- **cf**
    - talent : 재능(있는 사람)
    - talented : 재능이 있는

## dome : 집

- **domini** : 지배자
    - dominant : 지배적인, 우세한
    - dominantly : 지배적으로
    - dominance : 우월
- **predominant** : 두드러진
    - predominantly : 대개, 대부분
- **dominate** : 지배하다
    - domination : 지배
- **domain** : 영토, 영역
- **domestic** : 가정의, 국내의
    - domesticate : 길들이다
    - domestication : 길들이기

## relieve : 경감(안도)시키다

- **relevant** : 적절한, 관련있는
    - relevantly : 적절하게
    - relevance : 적절함, 관련(성)
    - **irrelevant** : 무관한, 관계없는
    - irrelevance : 무관함
- **relieved** : 안도하는
    - reliever : 구원, 투수
    - pain reliever : 진통제
- **relief** : 완화, 안도, 구제 물자

<br>

## vac : 빈, van : 빈

- **vacant** : 비어있는, 멍한
    - vacantly : 멍하니
    - vacancy : 공석, 빈방
- **vacate** : 비우다
    - vacation : 방학, 휴가
- **vacuum** : 진공, 진공청소기
- **vanity** : 허영심
    - vanish : 사라지다
- **vain** : 헛된
    - in vain : 헛되이

## gno : 알다

- **ignore** : 무시하다
    - ignorant : 무지한
    - ignorantly : 무지하게
    - ignorance : 무지
- **diagnose** : 진단하다
    - diagnosis : 진단
- **recognize** : 인지(인정)하다
    - recognition : 인지, 인정

    - knowledge : 지식

## st : 서다(세우다) 

- **distant** : 먼
    - distantly : 멀리
    - distance : 거리 
- **instant** : 즉각적인, 즉석의, 순간

    - instance : 사례, 경우
