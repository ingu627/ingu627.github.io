## 기본패턴

- `-al` : 형용사(~의, ~인)
- `-al + -ly = -aly` : 부사(~하게)
- `-al + ity = -ality` : 명사(~성, ~임)
  - 성격, 특징
- `-al + -ize = -alize` : 동사(~하다, ~화하다)
- `-al + -ize + -ation = -alization` : 명사(~화) 

<br>

## leg : 법

- **legal** : (합)법적인
  - legally : (합)법적으로
  - legality : 합법
  - legalize : 합법화하다
  - legalization : 합법화
- **illegal** : 불법적인
  - illegally : 불법적으로
  - illegality : 불법
- **legislate** : 법률을 제정하다
  - legislation : 법률 제정, 입법
  - legislator : 입법자 
  - legistlative : 입법의
  - legislature : 입법부, 의회

## norm : 표준, 규범

- **normal** : 보통의, 정상의

  - normalize : 정상화하다
  - normalization : 정화화
- **abnormal** : 비정상적인
  - abnormally : 비정상적으로
  - abnormality : 비정상
- **enormous** : 거대한, 막대한
  - enormouly : 엄청나게
