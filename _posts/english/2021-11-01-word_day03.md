## 기본패턴

- **-able** : 형용사(~할 수 있는)
- **-able + =ly = -ably** : 부사(~하게, ~로)
- **-able + -ity = -ability** : 명사(~성, ~함)
- **-ible** : 형용사(~할 수 있는)
- **-ible + -ly = -ibly** : 부사(~하게, ~로)
- **-ible + -ity = -ibility** : 명사(~성, ~함)

<br>
<br>

## able : 할 수 있는

- **ably** : 능숙하게
  - ability : 능력
- **unable** : 할 수 없는
  - inability : 무능, 불능
- **enbale** : 할 수 있게 하다
- **disable** : 장애를 입히다, 사용할 수 없게 하다
  - disabled : 장애(불구)의
  - disability : 장애
- **rehabilitate** : (명예, 건강을) 회복시키다
  - rehabilitation : 재활, 회복

<br>
<br>

## stable : 안정된

- **stably** : 안정되게
  - stability : 안정(성)
- **unstable** : 불안정한
  - instability : 불안정
- **stabilize** : 안정시키다
  - stabilization : 안정
- **establish** : 설립(제정)하다
  - establishment : 기관, 성립

<br>
<br>

## cap[capt] : 잡다

- **capable** : 할 수 있는, 유능한
  - capably : 유능하게
  - capability : 능력
  - incapable : 못하는, 무능한
  - incapability : 무능
- **capacity** : 용량, 능력
  - incapacity : 무능력
- **captive** : 포로, 사로잡힌
  - captivity : 감금, 억류
  - captivate : 매혹하다
  - captivation : 매혹
- **capture** : 포획(포착)하다, 포획

<br>
<br>

## probe : 조사하다, 탐사선

- **probable** : 있을 것 같은, 개연성 있는
  - probably : 아마
  - probaility : 개연성
  - improbable : 있을 것 같지 않은
- **prove** : 증명하다, 드러나다
  - approve : 찬성하다, 승인하다
  - approval : 찬성, 승인
  - disapprove : 불찬성하다
  - disapproval : 불찬성
- **proof** : 증거, 증명, 견디는
- **improve** : 향상(개선)시키다
  - improvement : 향상

## reason : 이유, 까닭, 추론하다

- **reasonable** : 합리적인, 적당한
  - reasonably : 합리적으로, 상당히
  - reasonability : 합리성, 적합성
  - unreasonable : 불합리한
  - unreasonably : 불합리하게
- **reasoning** : 추론

<br>

## sue : 고소(소송)하다

- **suit** : 어울리다, 정장, 옷, 소송
- **suitable** : 적합한
  - suitably : 적합하게
  - suitability : 적합(성)
  - unsuitable : 부적합한
  - unsuitably : 부적합하게
- **suitor** : 소송인
- **pursue** : 쫓다, 추구하다
  - pursuit : 추격, 추구
- **cf**
  - in pursuit of : ~을 추구하여

## fort : 힘

- **comfort** : 편안, 위로, 위로하다
  - comfortable : 편안한
  - comfortably : 편안하게
  - uncomfortable : 불편한
  - uncomfortably : 불편하게 
  - discomfort : 불편, 불편하게 하다
- **effort** : 노력
- **fortify** : 강화하다
  - fortitude : 용기, 꿋꿋함
  - fortress : 요새
- **force** : 힘, (미)군, 강요하다
  - forceful : 강력한
  - enforce : 시험(집행)하다
  - reinforce : 보강하다, 강화하다
  - reinforcement : 요강, 강화

## poss[pot] : 힘이 있다

- **possible** : 가능한
  - possibly : 아마, (부정) 도저히

  - impossible : 불가능한
  - impossibly : 불가능하게
  - impossibility : 불가능(성)

## cred : 믿다

- **credible** : 믿을 수 있는
  - credibly : 확실히
  - credibility : 신뢰성
  - incredible : 믿을 수 없는
  - incredibly : 믿을 수 없게, 엄청나게
  - incredibility : 믿을 수 없음
