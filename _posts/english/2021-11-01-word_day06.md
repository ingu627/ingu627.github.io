## 기본패턴

- **-ful** : 형용사(~이 가득한)
- **-ful + -ly = -fully** : 부사(~하게, ~로)
- **-ful + -ness = -fulness** : 명사(~한 상태, 성질)

<br>
<br>

## full : 가득한 완전한

- fully : 완전히
- fullness : 충분함, 풍만함
- full-time : 정규직의
- **fill** : 채우다(with)
  - fulfill : 이루다, 이행하다
  - landfill : 매립(지)
- **refill** : 다시 채우다, 리필
- **cf**
  - fill out : (서적 등의) 빈 곳을 채우다
  - fill up : 가득 채우다


<br>
<br>

## use : 쓰다, 이용(사용)하다, 이용, 사용

- **useful** : 유용한
  - usefully : 유용하게
  - usefullness : 유용성
- **useless** : 쓸모없는
  - uselessly : 쓸데없이
- **user** : 사용자
- **usual** : 흔히 하는, 보통의
  - usually : 보통, 대개
  - as usual : 평소처럼
- **unusual** : 특이한, 드문
  - unusually : 대단히

<br>
<br>

## care : 주의, 돌봄, 상관하다

- **careful** : 주의하는, 조심스런
  - carefully : 주의하여, 조심스럽게
  - carefulness : 조심성
- **careless** : 부주의한
  - carelessly : 부주의하게
  - carelessness : 부주의
- **cf**
  - caretaker : 관리인
  - caregiver : 돌보는 사람
  - health care : 건강관리, 의료
  - medicare : 노인 의료보험
- **cure** : 치료하다, 치료범
  - curable : 치료할 수 있는
  - incurable : 불치의, 불치환자

<br>
<br>

## power : 힘, 권력, 동력

- **powerful** : 강력한, 영향력 있는
  - powerfully : 강력하게
- **powerless** : 무력한
- **powered** : 동력의
- **cf**
  - willpower : 의지력
  - horsepower : 마력
  - manpower : 인력
- **empower** : 권한을 주다, 할 수 있게 하다
  - empowerment : 권한부여

<br>
<br>

## pen : 형벌

- **pain** : 고통, 아픔
  - painful : 고통스런, 아픈
  - painfully : 고통스럽게
- **painless** : 고통없는
- **painstaking** : 공들인, 힘든
- penal : 형벌의
- penalty : 처벌, 벌(금) 
- punish : 처벌하다
- punishment : 처벌

<br>
<br>

## fy : 믿다

- **faith** : 믿음, 신념
  - faithful : 충실한
  - unfaithful : 충실하지 않은, 부정한
  - faithfully : 충실하게
  - faithless : 믿을 수 없는, 부정한
- **defy** : 반항(무시)하다
  - defiant : 반항하는
  - defiantly : 반항적으로
  - defiance : 반항, 무시
  - in defiance of : ~에 대항하여
- **fiance** : 약혼자
  - fiance : 약혼녀

<br>
<br>

## truth : 진실, 사실

- **truthful** : 진실한, 정직한
  - truthfully : 정직하게
  - untruthful : 거짓의
- **true** : 진짜의, 사실의
  - truly : 진실로, 참으로
- **trust** : 신뢰, 신뢰하다
  - mistrust : 불신, 불신하다
  - distrust : 불신, 불신하다
- **trustworthy** : 신뢰할 수 있는

<br>
<br>

## sucess : 성공

- **sucessful** : 성공한
  - sucessfully : 성공적으로
  - unsucessful : 실패한
  - unsuccessfully : 실패하여
- **succeed** : 뒤를 잇다(to), 성공하다(in)
- **cf**
  - succession : 연속, 승계
  - successive : 연속적인
  - successor : 후계자

<br>
<br>

## skel : 자르다

- **skill** : 기술, 기량
  - skillful : 능숙한
  - skillfully : 능숙하게
  - skilled : 숙련된
- **shelf** : 선반
  - shelve : 선반에 두다, 보류하다
- **sculpt** : 조각하다
  - sculptor : 조각가
  - sculpture : 조각(품)
- scalpel : (수술용)메스

<br>
<br>

## right : 옳은, 오른쪽의, 즉시, 바로, 권리, 오른쪽

- **rightful** : 정당한, 적법한
  - rightfully : 정당하게
- **righteous** : 정의로운, 올바른
  - righteously : 정의롭게, 올바르게
  - righteousness : 정의
- **cf**
  - upright : 수직의, 꼿꼿한
  - downright : 완전한
  - forthright : 솔직한
  - copyright : 저작권
  - human right : 인권

<br>
<br>

## hand : 손, 건네다

- **handful** : 한줌, 소량
  - a handful of : 한줌의, 소수의
- **handle** : 다루다, 처리하다, 손잡이
- **cf**
  - handicap : 핸디캡, 장대
  - handcuff : 수갑
  - handkerchief : 손수건
  - handshake : 악수
- **cf**
  - hand dawn : 후세에 전하다
  - hand in : 제출하다
  - hand over : 양도하다, 넘겨주다
  - hand out : 배포하다, 나눠주다
- **cf**
  - first-hand : 직접적인
  - second-hand : 간접적인
  - at hand : 가까이에
  - off hand : 즉석에서
  - on the other hand : 다른 한편으로
  - out of hand : 통제 할 수 없는
